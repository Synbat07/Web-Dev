This is a simple OOP project in Python about flowers.

I made:
- 1 base class: Flower
- 3 child classes: Rose, Lily, Lavender

Flower has 4 attributes:
- name
- color
- price
- smell

Flower has methods:
- bloom()
- get_info()
- __str__()

Child classes have their own methods:
- Rose -> give()
- Lily -> decorate()
- Lavender -> relax()

I used inheritance.
I also overrode the bloom() method.
In main.py I created objects, put them in a list, used a loop and printed methods.
